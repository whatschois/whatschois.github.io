# What's Choi's

> Keep in Trends

Personal blog built with [Jekyll](https://jekyllrb.com/) and the
[Chirpy][chirpy] theme, deployed to GitHub Pages.

Live at: <https://whatschois.github.io>

## Local development

```bash
bundle install
npm install
bundle exec jekyll serve
```

## Deployment

Pushing to `master` triggers `.github/workflows/pages-deploy.yml`, which
builds the site and publishes it to GitHub Pages.

[chirpy]: https://github.com/cotes2020/jekyll-theme-chirpy
